3. **`BskyClient::new(credentials: Option<(&str, &str)>) -> Self`:**
   - If credentials is None: set `base_url = PUBLIC_API_BASE`, emit `tracing::warn!("No BSKY_APP_PASSWORD set; using public API with lower rate limits")`
   - If credentials is Some: set `base_url = PDS_BASE`
   - Create `reqwest::Client::new()`

4. **`async fn authenticate(&mut self, handle: &str, password: &str) -> Result<String, ExtractorError>`:**
   - POST to `/xrpc/com.atproto.server.createSession` with `CreateSessionRequest { identifier: handle, password }`
   - Deserialize response as `CreateSessionResponse`
   - Store `access_jwt` and `refresh_jwt` in self
   - Return `did` from response
   - On non-2xx: return `ExtractorError::AuthFailed(status + body text)`

5. **`async fn refresh_auth(&mut self) -> Result<(), ExtractorError>`:**
   - POST to `/xrpc/com.atproto.server.refreshSession` with Bearer `self.refresh_jwt`
   - On success: update `self.access_jwt` and `self.refresh_jwt`
   - On failure: return `ExtractorError::AuthExpired`

6. **`async fn resolve_handle(&self, handle: &str) -> Result<String, ExtractorError>`:**
   - GET `/xrpc/com.atproto.identity.resolveHandle?handle={handle}`
   - Deserialize as `ResolveHandleResponse`
   - Return `did`
   - On error: `ExtractorError::InvalidHandle(handle.to_string())`

7. **`async fn execute(&mut self, method: reqwest::Method, path: &str, query: &[(&str, &str)], body: Option<&[u8]>) -> Result<bytes::Bytes, ExtractorError>`:**
   - Retry loop up to `MAX_RETRIES`:
     - Build request with method, `self.build_url(path)`, query params
     - If `self.access_jwt` is Some, add `Authorization: Bearer {token}` header
     - If body is Some, set Content-Type application/json and body
     - On 429: parse `Retry-After` header (seconds), fall back to `ratelimit-reset` header (Unix epoch), fall back to `backoff_delay(attempt)`. Sleep and increment attempt.
     - On 401 and attempt == 0: call `self.refresh_auth().await`, retry (do not increment attempt counter, just set a "refreshed" flag to prevent double-refresh)
     - On 401 after refresh: return `ExtractorError::AuthExpired`
     - On success (2xx): return response bytes
     - On timeout/connect error: sleep `backoff_delay(attempt)`, retry
     - On other error: return `ExtractorError::Http(status)`
   - After MAX_RETRIES: return `ExtractorError::RateLimitExhausted`

8. **`fn backoff_delay(attempt: u32) -> std::time::Duration`:**
   - `Duration::from_secs(1u64.saturating_shl(attempt).min(60))`
   - Public for testing

9. **`fn parse_retry_after(headers: &reqwest::header::HeaderMap) -> Option<std::time::Duration>`:**
   - Check `Retry-After` header first (parse as u64 seconds)
   - Fall back to `ratelimit-reset` header (parse as Unix epoch, subtract current time)
   - Return None if neither present
   - Public for testing

10. **`fn build_url(&self, path: &str) -> String`:**
    - `format!("{}{path}", self.base_url)`

11. **Helper `fn extract_post_fields(value: &serde_json::Value) -> Option<(String, String)>`:**
    - Extract `value["text"]` as String, `value["createdAt"]` as String
    - Return None if either field missing
    - Public for testing
